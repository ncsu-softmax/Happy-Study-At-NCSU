## 📖 关于本项目

这个仓库旨在为NCSU的学生、教职工以及RTP地区的居民提供一个全面的生活指南。无论你是刚来的新生，还是在这里生活多年的老居民，都欢迎你探索、贡献和分享这里的美好！

## 🤝 如何贡献

我们欢迎所有的贡献！如果你发现了新的好地方，或者想要更新现有信息：

1. Fork 这个仓库
2. 创建你的特性分支 (`git checkout -b feature/AmazingPlace`)
3. 提交你的更改 (`git commit -m 'Add some AmazingPlace'`)
4. 推送到分支 (`git push origin feature/AmazingPlace`)
5. 开启一个 Pull Request

### 贡献格式示例
```markdown
### 类别名称
- **地点名称** - 详细地址 | 特色信息 | 价格区间
  - 详细描述，包括个人体验、推荐菜品/项目、最佳访问时间等
  - 📍 Google Maps 链接
  - 🕒 营业时间
  - 💰 人均消费
  - ⭐ 个人评分
```
